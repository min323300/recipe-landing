## 2. 이미지 컬럼 추가하기

### 📌 각 시트에 `이미지URL` 컬럼 추가

**카테고리 시트** (C열에 추가)
| 카테고리명 | 아이콘 | **이미지URL** |
|------------|--------|---------------|
| 찌개/국 | 🍲 | images/category-jjigae.png |

**레시피 시트** (H열에 추가)
| ID | 제목 | 설명 | 카테고리 | 유튜브링크 | 블로그링크 | 이모지 | **이미지URL** |
|----|------|------|----------|------------|------------|--------|---------------|
| 1 | 김치찌개 | 10분 완성 | 찌개/국 | ... | ... | 🍲 | images/recipe-kimchi.png |

**상품 시트** (G열에 추가)
| 레시피ID | 상품명 | 가격 | 이모지 | 쇼핑몰 | 쇼핑링크 | **이미지URL** |
|----------|--------|------|--------|--------|----------|---------------|
| 1 | 종가집 김치 | 12,900원 | 🥬 | 쿠팡 | ... | images/product-kimchi.png |

> 💡 **이미지URL이 없으면 이모지가 표시됩니다!**

## 3. GitHub에 이미지 업로드

### 방법 1: GitHub 웹에서 직접 업로드 (추천 ⭐)

**1단계**: GitHub 레포지토리 접속
```
https://github.com/min323300/recipe-landing
```

**2단계**: images 폴더 만들기
1. `Add file` → `Create new file` 클릭
2. 파일명에 `images/.keep` 입력
3. `Commit new file` 클릭

**3단계**: 이미지 업로드
1. `images` 폴더 클릭
2. `Add file` → `Upload files` 클릭
3. 이미지 파일 드래그 앤 드롭
4. `Commit changes` 클릭

### 방법 2: GitHub Desktop 사용

1. 레포지토리 클론
2. `images` 폴더 생성
3. 이미지 파일 복사
4. Commit & Push

## 4. 구글 시트에 이미지 연결

### 📌 이미지 URL 형식

**GitHub 이미지 경로**
```
images/파일명.png
```

예시:
| 용도 | 이미지URL |
|------|-----------|
| 카테고리 | images/category-jjigae.png |
| 레시피 | images/recipe-kimchi.png |
| 상품 | images/product-kimchi.png |

### 📌 파일명 규칙

✅ **좋은 예**
- `kimchi-jjigae.png`
- `recipe_01.jpg`
- `product-tofu.png`

❌ **나쁜 예**
- `김치찌개.png` (한글 X)
- `recipe 01.jpg` (공백 X)
- `product@tofu.png` (특수문자 X)

---

## 5. 전체 작업 순서

```
1. GitHub에 images 폴더 생성
      ↓
2. 이미지 파일 업로드 (영문 파일명)
      ↓
3. 구글 시트에 이미지URL 컬럼 추가
      ↓
4. 이미지 경로 입력 (images/파일명.png)
      ↓
5. 사이트 새로고침 → 이미지 표시! 🎉
```

---

## 6. 동작 방식

| 상황 | 결과 |
|------|------|
| 이미지URL 있음 | 이미지 표시 |
| 이미지URL 없음 | 이모지 표시 |
| 이미지 로드 실패 | 이모지로 대체 |

**이모지만 사용해도 사이트는 정상 작동합니다!**
이미지는 나중에 천천히 추가하셔도 됩니다.

## 🆘 문제 해결

### 이미지가 안 보여요
1. 파일명에 한글/공백 없는지 확인
2. 경로가 `images/파일명.png` 형식인지 확인
3. GitHub에 파일이 업로드됐는지 확인
4. 브라우저 캐시 삭제 후 새로고침

### 데이터가 안 나와요
1. 구글 시트 "웹에 게시" 확인
2. 시트 이름 확인 (카테고리, 레시피, 상품)
3. 컬럼명 정확한지 확인
