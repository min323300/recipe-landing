# 🍳 레시피 랜딩페이지 - 구글 시트 연동 버전

구글 시트에서 **카테고리, 레시피, 상품**을 모두 관리할 수 있습니다!
코드 수정 없이 시트만 편집하면 사이트에 바로 반영됩니다.

---

## 📊 구글 시트 설정 방법

### Step 1: 구글 시트 만들기

1. [구글 시트](https://sheets.google.com) 접속
2. **+ 빈 스프레드시트 만들기** 클릭
3. 시트 이름을 `레시피 관리` 등으로 변경

---

### Step 2: 시트 3개 만들기

하단 탭에서 시트 3개를 만드세요:

| 시트 이름 | 용도 |
|----------|------|
| `카테고리` | 카테고리 관리 |
| `레시피` | 레시피 정보 |
| `상품` | 쿠팡/쇼핑 상품 |

---

### Step 4: 시트 웹에 게시하기

1. 구글 시트에서 **파일** → **공유** → **웹에 게시** 클릭
2. **전체 문서** 선택
3. **게시** 버튼 클릭
4. "게시하시겠습니까?" → **확인**

---

### Step 5: 시트 ID 복사하기

구글 시트 URL에서 ID를 복사하세요:

```
https://docs.google.com/spreadsheets/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/edit
                                      ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
                                      이 부분이 시트 ID입니다
```

---

### Step 6: index.html에 시트 ID 입력

`index.html` 파일을 열고 상단의 설정 부분을 수정하세요:

```javascript
// 🔧 설정 - 여기만 수정하세요!
const SHEET_ID = '1AbCdEfGhIjKlMnOpQrStUvWxYz';  // ← 여기에 시트 ID 붙여넣기
```

---

### Step 7: GitHub에 업로드

수정한 `index.html`을 GitHub 레포지토리에 업로드하면 끝!

## ✅ 관리 방법

### 카테고리 추가하기
1. 구글 시트 `카테고리` 탭 열기
2. 새 행에 카테고리명, 아이콘 입력
3. 저장하면 사이트에 바로 반영!

### 레시피 추가하기
1. 구글 시트 `레시피` 탭 열기
2. 새 행에 ID, 제목, 설명, 유튜브링크, 티스토리링크 등 입력
3. 저장하면 사이트에 바로 반영!

### 상품 연결하기
1. 구글 시트 `상품` 탭 열기
2. 새 행에 레시피ID, 상품명, 가격, 쇼핑링크 입력
3. 저장하면 사이트에 바로 반영!

## ❓ 문제 해결

### "데이터를 불러올 수 없습니다" 오류
1. 시트가 **웹에 게시**되었는지 확인
2. 시트 ID가 올바른지 확인
3. 시트 이름이 정확한지 확인 (카테고리, 레시피, 상품)

### 상품이 안 보여요
- `상품` 시트의 `레시피ID`가 `레시피` 시트의 `ID`와 일치하는지 확인
